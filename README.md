# Switchboard

## Software requirements

```bash
sudo apt-get install python-dev
sudo pip install Flask
sudo pip install flask-hmacauth
```

## Components

 * Raspberry Pie
 * 5v relay module
 * 10 pin female to female breadboard jumper wires

Examples:
